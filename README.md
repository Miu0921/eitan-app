# eitan-app

## 概要
このプロジェクトは語彙学習アプリケーションです。

## インストール方法
1. リポジトリをクローンします。
    ```bash
    git clone https://github.com/Miu0921/eitan-app.git
    ```
2. 必要な依存関係をインストールします。
    ```bash
    npm install
    ```

## 使い方
1. アプリケーションを起動します。
    ```bash
    npm start
    ```
2. ブラウザで`http://localhost:3000`にアクセスします。
